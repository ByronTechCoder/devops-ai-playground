# devops-ai-playground

A monorepo for learning and experimenting with DevOps, AI agent creation, AI development, and coding projects with AI.

## Repository Structure

```
devops-ai-playground/
├── devops/          # DevOps experiments, pipelines, infrastructure-as-code
├── ai-agents/       # AI agent creation and experiments
├── ai-projects/     # AI development projects and notebooks
├── coding-projects/ # General coding projects enhanced with AI
└── docs/            # Shared documentation and learning notes
```

## Sections

### 🛠️ DevOps (`devops/`)
Experiments with CI/CD pipelines, Docker, Kubernetes, Terraform, monitoring, and other DevOps tooling. Learn infrastructure-as-code and automation practices.

### 🤖 AI Agents (`ai-agents/`)
Projects focused on building and experimenting with AI agents — autonomous systems that can reason, plan, and act to accomplish tasks using LLMs and tool-use frameworks.

### 🧠 AI Projects (`ai-projects/`)
Broader AI/ML development including fine-tuning, RAG pipelines, embeddings, prompt engineering, and integrations with AI APIs (OpenAI, Anthropic, etc.).

### 💻 Coding Projects (`coding-projects/`)
General software projects that incorporate AI — from AI-assisted code generation experiments to apps built with AI-powered features.

## Getting Started

Each subdirectory contains its own `README.md` with setup instructions specific to that project or experiment. Browse to a section that interests you and follow the instructions there.

## Contributing

This is a personal learning playground. Feel free to fork and adapt it for your own experiments!
