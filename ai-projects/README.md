# AI Projects

This section contains broader AI/ML development projects — from prompt engineering experiments to RAG pipelines and model fine-tuning.

## Topics Covered

- **Prompt Engineering** — Systematic prompt design, few-shot learning, chain-of-thought
- **RAG (Retrieval-Augmented Generation)** — Building knowledge-grounded AI applications
- **Embeddings** — Vector databases, semantic search, similarity matching
- **Fine-tuning** — Adapting pre-trained models to custom tasks
- **AI APIs** — Integrations with OpenAI, Anthropic, Hugging Face, and others
- **Notebooks** — Jupyter notebooks for exploratory AI/ML experiments

## Directory Layout

```
ai-projects/
├── prompt-engineering/   # Prompt design experiments
├── rag/                  # Retrieval-augmented generation pipelines
├── embeddings/           # Embedding and vector search experiments
├── fine-tuning/          # Model fine-tuning experiments
└── notebooks/            # Jupyter notebooks for exploration
```

## Getting Started

Python is the primary language for this section.

```bash
python3 -m venv .venv
source .venv/bin/activate   # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

For Jupyter notebooks:

```bash
pip install jupyter
jupyter notebook
```
