# Coding Projects

This section contains software projects that incorporate AI — from AI-assisted development experiments to applications with AI-powered features.

## Topics Covered

- **AI-assisted development** — Using Copilot, Cursor, or similar tools in real projects
- **AI-powered apps** — Building applications that surface AI capabilities to users
- **Automation scripts** — Scripts that use LLMs for text processing, summarization, etc.
- **Integrations** — Connecting AI APIs into existing tools and workflows
- **Web & CLI apps** — Practical apps built with an AI-first approach

## Directory Layout

```
coding-projects/
├── scripts/         # Utility scripts leveraging AI APIs
├── web-apps/        # Web applications with AI features
├── cli-tools/       # Command-line tools powered by AI
└── integrations/    # AI integrations with third-party services
```

## Getting Started

Projects in this section may use a variety of languages and runtimes. Each subdirectory has its own `README.md` with specific setup instructions. Common setups:

**Python:**
```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

**Node.js:**
```bash
npm install
```

Copy `.env.example` to `.env` and fill in any required API keys before running.
